# cz2swf
Bash/C++ utility to create flash (.swf) files from .cz files using http://www.libming.org/

The .cz format was created to save flash projects done with Cazabe. The cz2swf tool outputs a .swf movie from a .cz file.

To install cz2swf refer to the "INSTALL" file

For more details visit https://0xdac.org/cazabe.html
